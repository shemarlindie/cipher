# Cipher

## Project Description
A simple substitution cipher tool. Ciphers text using a Caesar or Polyalphabetic cipher.

![](./Screenshots/cipher.png)

## Features:
* Caesar Cipher
* Polyalphabetic Cipher

## Requirements:
* Microsoft's .NET Framework 4.5 or later (comes with most Windows systems)
